Nama : Syihwa Moza Alika YP Kastella

## Deskripsi
Project ini merupakan implementasi REST API CRUD data KTP menggunakan Spring Boot dan MySQL. 
Aplikasi ini memungkinkan pengguna untuk menambah, melihat, mengubah, dan menghapus data KTP melalui API dan website sederhana menggunakan HTML, CSS, JavaScript, dan JQuery Ajax.
